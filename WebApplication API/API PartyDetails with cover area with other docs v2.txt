


For Add New party Info:
API:
Create a new POST request with the URL:

https://asianabpolymer.com/abpolymer/salesforce_api.php?action=create_party

Set the headers:
Content-Type: application/json
Set the Body: Select raw and choose JSON. Enter the following JSON data:


{
    "PartyName": "ABC Distributors",
    "Address": "123 Main Street, Dhaka",
    "RegionID": 1,
    "OwnerName": "John Doe",
    "OwnerCurrentAddress": "Dhaka City",
    "OwnerPermanentAddress": "Chittagong",
    "OwnerContactNumber": "017XXXXXXXX",
    "OwnerDateOfBirth": "1985-05-15",
    "BusinessStartYear": "2010",
    "Email": "owner@example.com",
    "Website": "http://www.abcdistributors.com",
    "ContactPersonName": "Jane Smith",
    "ContactNumber": "018XXXXXXXX",
    "NoOfThana": 5,
    "NoOfDistrict": 3,
    "CreditLimit": 500000.00,
    "DepositAmount": 100000.00,
    "IsSamityMember": "Yes",
    "WayOfSendingLetters": "Email",
    "OpeningBalance": 20000.00
}

Output
{
    "message": "Party created successfully"
}

For Update party Info:
API:
create a PUT request.

https://asianabpolymer.com/abpolymer/salesforce_api.php?action=update_party&PartyID=59

Go to the Body tab, select raw, and set it to JSON.
Enter the following JSON in the body

{
    "PartyName": "Updated DistributorsSalim",
    "Address": "456 New Street, Dhaka",
    "RegionID": 2,
    "OwnerName": "John Doe Updated",
    "OwnerCurrentAddress": "New Dhaka",
    "OwnerPermanentAddress": "Chittagong Updated",
    "OwnerContactNumber": "018XXXXXXXX",
    "OwnerDateOfBirth": "1985-06-15",
    "BusinessStartYear": "2012",
    "Email": "updated@example.com",
    "Website": "http://www.updateddistributors.com",
    "ContactPersonName": "Jane Doe Updated",
    "ContactNumber": "019XXXXXXXX",
    "NoOfThana": 6,
    "NoOfDistrict": 4,
    "CreditLimit": 700000.00,
    "DepositAmount": 150000.00,
    "IsSamityMember": "No",
    "WayOfSendingLetters": "Courier",
    "OpeningBalance": 30000.00
}

Output:

{
    "message": "Party updated successfully"
}

For Party CoveredArea Create

Create a new POST request with the URL:

https://asianabpolymer.com/abpolymer/salesforce_api.php?action=create_partydetailsAreas&PartyID=1

Set the headers:
Content-Type: application/json
Set the Body: Select raw and choose JSON. Enter the following JSON data:

Input:

{
  "PartyAreas": [
    {
      "RegionID": 1,
      "RegionName": "North Region"
    },
    {
      "RegionID": 2,
      "RegionName": "South Region"
    }
  ]
}


Output:

{
  "message": "Party Area created successfully",
  "PartyID": "<PartyID>"
}


For Update update_partydetailsAreasInfo:
API:
create a PUT request.

https://asianabpolymer.com/abpolymer/salesforce_api.php?action=update_partydetailsAreas&PartyID=1

Go to the Body tab, select raw, and set it to JSON.
Enter the following JSON in the body

Input:

{
  "PartyAreas": [
    {
      "RegionID": 1,
      "RegionName": "Updated Region A"
    },
    {
      "RegionID": 3,
      "RegionName": "New Region C"
    }
  ]
}

Output:

{
    "message": "Party Areas updated successfully",
    "PartyID": 1
}
