Method: POST

URL:

vbnet
Copy code
https://asianabpolymer.com/abpolymer/salesforce_api.php?action=update_partydetailsAndroid&PartyID=1
Replace:

your-server-domain with your server's address (e.g., https://asianabpolymer.com/abpolymer).
path-to-api with the API endpoint path.
123 with the partyID you want to update.
Headers:

Content-Type: multipart/form-data
Body:
Set the body type to form-data and include the following fields:

Form Data
Key	Value
PartyName	Updated Party Name (e.g., "Test Party Updated")
Address	Updated Address (e.g., "123 Test Street")
RegionID	Updated Region ID (e.g., 5)
OwnerName	Owner's name (optional)
OwnerCurrentAddress	Owner's current address (optional)
OwnerPermanentAddress	Owner's permanent address (optional)
OwnerContactNumber	Owner's contact number (optional)
OwnerDateOfBirth	Owner's date of birth (YYYY-MM-DD) (optional)
BusinessStartYear	Business start year (optional)
Email	Updated email address (optional)
Website	Updated website URL (optional)
ContactPersonName	Updated contact person name (optional)
ContactNumber	Updated contact person's phone number (optional)
NoOfThana	Updated number of Thanas (optional)
NoOfDistrict	Updated number of districts (optional)
CreditLimit	Updated credit limit (optional)
DepositAmount	Updated deposit amount (optional)
IsSamityMember	Is a Samity member? (1 for yes, 0 for no)
WayOfSendingLetters	Way of sending letters (optional)
OpeningBalance	Updated opening balance (optional)
OwnerPicturePath (file)	Upload a new image for the owner picture (optional)
OwnerNIDDocumentPath (file)	Upload a new NID document file (optional)
TradeLicenseDocumentPath (file)	Upload a new trade license document file (optional)
SamityMembershipCardPath (file)	Upload a new Samity membership card file (optional)
TINCertificatePath (file)	Upload a new TIN certificate file (optional)
DepositChequePath (file)	Upload a new deposit cheque file (optional)
NonJudicialAgreementDocumentPath (file)	Upload a new non-judicial agreement document (optional)
AgreementDocumentPath (file)	Upload a new agreement document file (optional)

For Party CoveredArea Create

Create a new POST request with the URL:

https://asianabpolymer.com/abpolymer/salesforce_api.php?action=create_partydetailsAreas&PartyID=1

Set the headers:
Content-Type: application/json
Set the Body: Select raw and choose JSON. Enter the following JSON data:

Input:

{
  "PartyAreas": [
    {
      "RegionID": 1,
      "RegionName": "North Region"
    },
    {
      "RegionID": 2,
      "RegionName": "South Region"
    }
  ]
}


Output:

{
  "message": "Party Area created successfully",
  "PartyID": "<PartyID>"
}


For Update update_partydetailsAreasInfo:
API:
create a PUT request.

https://asianabpolymer.com/abpolymer/salesforce_api.php?action=update_partydetailsAreas&PartyID=1

Go to the Body tab, select raw, and set it to JSON.
Enter the following JSON in the body

Input:

{
  "PartyAreas": [
    {
      "RegionID": 1,
      "RegionName": "Updated Region A"
    },
    {
      "RegionID": 3,
      "RegionName": "New Region C"
    }
  ]
}

Output:

{
    "message": "Party Areas updated successfully",
    "PartyID": 1
}

Key	Value	Type
OtherDocs[0][PartyOtherDocName]	Document1	Text
OtherDocs[1][PartyOtherDocName]	Document2	Text
Document1	Choose a file	File
Document2	Choose another file	File
