2. Get Parties with Search Filter
Request:

Method: GET

URL: https://asianabpolymer.com/abpolymer/salesforce_api.php?action=get_parties&search=ABC

Response: Returns only parties containing "ABC" in PartyName, PartyCode, ContactName, or ContactNumber

3. Get Parties with Status Filter
Request:

Method: GET

URL: https://asianabpolymer.com/abpolymer/salesforce_api.php?action=get_parties&status=Active

Response: Returns only active parties

4. Get Parties with District Filter
Request:

Method: GET

URL: https://asianabpolymer.com/abpolymer/salesforce_api.php?action=get_parties&district_id=10

Response: Returns only parties from DistrictID 10

5. Get Parties with Pagination
Request:

Method: GET

URL: https://asianabpolymer.com/abpolymer/salesforce_api.php?action=get_parties&page=2&limit=10

Response: Returns page 2 with 10 records per page
