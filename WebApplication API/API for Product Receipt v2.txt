Create Receipt all

https://asianabpolymer.com/abpolymer/salesforce_api.php?action=create_ppreceiptall

1. Set Up the Request
Method: POST
URL: Enter the endpoint URL where the create_ppreceiptall function is hosted, for example:
arduino
Copy code


2. Configure Headers
Add the following headers if needed:
Key: Content-Type
Value: multipart/form-data
3. Add the Body
Mode: form-data
Add the following fields in the Key column of the form-data tab:
Main Product Receipt Data:
Key	Value (Example)	Type
ProductReceiptNo	PR-001	Text
ReceiptDate	2024-12-21	Text
BindingPartyID	101	Text
PrintEdition	First Edition	Text (optional)
ChallanNumber	CH-2024-01	Text (optional)
ProductionOrderQty	500	Text
UserID	1	Text
Challan Copy Path (File Upload):
Key	Value (Example)	Type
ChallanCopyPath	Choose a file (PDF/Image)	File
Product Receipt Details (JSON Array):
Key	Value (Example)	Type
Details	[{"FinancialYearID":1, "ProductCategoryID":2, "ProductID":3, "Quantity":100, "Rate":50.25}]	Text
4. Send the Request
Click Send in Postman.
5. Analyze the Response
Check the response from the API in the response tab:

Success Response
json
Copy code
{
    "message": "Product receipt and details created successfully",
    "ProductReceiptID": 123,
    "ChallanCopyPath": "https://asianabpolymer.com/abpolymer/uploads/ChallanCopyPath/Challan_676917ef6d2ee5.11568405.jpg"
}
