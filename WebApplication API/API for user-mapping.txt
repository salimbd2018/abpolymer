18.	Mapping Employee VS Region
Note: 
1.	Load all the employee left site with radio button and loan all the Region like Tree with Check box
2.	Here we will select one-to-many system.
The data can save & edit. No delete option.
SQL: 
select id, UserID, RegionID from sndMapEmpRegion
INSERT INTO [dbo].[sndMapEmpRegion]
           ([UserID]
           ,[RegionID])
     VALUES
           (<UserID, int,>
           ,<RegionID, int,>)
