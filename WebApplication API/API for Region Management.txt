4. RegionTypeID PUT Request - Update Region: RegionTypeID

How It Works
When you call update_region, it:
Deletes the existing region.
Adds a new entry with the updated data.
Returns the RegionID of the new entry.

   - URL: https://asianabpolymer.com/abpolymer/salesforce_api.php?action=update_region&RegionID=1
   - Method: `PUT`
   - Headers: Content-Type: application/json
   - Body (JSON):
     ```json
 
 {
  "RegionName": "Updated Region Name",
  "RegionTypeID": 90,
  "ParentRegionID": 5
}

Output:

{
  "message": "Region updated successfully",
  "NewRegionID": 1234
}
